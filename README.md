# My Task Tracker

A simple task management web app built with React.

🔗 **Live Demo**: [my-tasks-tracker.vercel.app](https://my-tasks-tracker.vercel.app)

## Features

- Add new tasks
- Mark tasks as complete (click on task text)
- Delete tasks
- Time tracking of each task
- Data persistence with localStorage (tasks survive page refresh)

## Tech Stack

- React (useState hooks)
- JavaScript
- CSS (inline styles)
- Vercel (deployment)
